Below are the frameworks and compilers we used to build this project .

Handlebars

Written in JavaScript, Handlebars. js is a compiler that takes any HTML and Handlebars expression and compiles them to a JavaScript function We started by including the hbs generator in our folder and used it to create layouts for this project.

Express

Before using handlebars,we had downloaded node and learnt how to import express in node, learning also about package.json file and how it stores dependencies.Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

Bootstrap

Before using bootstrap in our layouts, we included the links of the bootstrap in the head section of the layout page in views folder of hbs generator.Bootstrap is a HTML, CSS, and JavaScript framework for developing responsive websites.
